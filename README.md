IDXExchange Data Analyst Internship Project Repo

# Project Structure Overview

This repository contains the following main folders and files. Each section is briefly described below:

- **Notebook/**  
	Includes Jupyter Notebook files (such as 01_EDA.ipynb, 02_dataset_cleaning.ipynb) for interactive data exploration, cleaning, analysis, and visualization.

- **src/**  
	Main project code directory, including the following submodules:
	- **analysis/**  
		Scripts for data analysis, such as missing value handling.
	- **data_functions/**  
		Reusable data processing function modules, such as data merging and feature engineering.
	- Other Python scripts (e.g., load_data.py) for data loading and related tasks.

- **Weekly_Deliverable/**  
	Weekly deliverable Python scripts, documenting weekly progress and analysis results.
